## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/Ahmed-Mohamed-Saeed/alpha-app.git
```

## Tech Stack

- React Native
- Expo
- UI Animated
- NativeWind (Tailwind CSS for React Native)

## How to Run

2. Install dependencies

   ```bash
   npm install or yarn
   ```

3. Start the app

   ```bash
   npx expo start
   ```

## Get a fresh project

When you're ready, run:

```bash
npm run reset-project
```
